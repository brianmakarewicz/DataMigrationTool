Object Detail — record breakdown (funnel redesign)

One object in one run: breadcrumb back up the chain, job chips, log link, and the record breakdown (funnel). Click any count → Record List filtered to those rows. Click a job chip → ESS Job Detail. Click View Log → Activity Log browser, filtered to this work item.

Drilling from a run tile (Page 82) into an object lands here. Everything stays on one page — no extra drill level. The record breakdown becomes a horizontal funnel table: one row per record type in the FBDI, stages as columns reading left→right, the drop at each stage as a small red count between stages. Every number is a clickable pill filtering Page 57 to exactly those rows.
